Courses This Semester

Web Technology and Concepts

This course focuses on the basics of web development, including how websites are designed and built. It covers topics like HTML, CSS, JavaScript, and the general concepts of how the internet works, preparing me to create and manage websites.

C++ Programming

In this course, I am learning the fundamentals of object-oriented programming using C++. Topics include variables, control structures, functions, classes, inheritance, and polymorphism, which are essential for software development.
